# Server

Server application.

1. Install Python
  - https://www.python.org/downloads/
  - Ensure "Add to PATH" is checked for Windows
  - verify with: python -v
2. Install pip (Package/library manager for python)
  - python get-pip.py
  - verify with: pip -V
3. Install dependencies
  - pip -r requirements.txt
4. Run locally
  - python main.py
